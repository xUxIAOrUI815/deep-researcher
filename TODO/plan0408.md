# 0408 需求

## 整体目标和职责边界

### graph.py

负责：
    - 节点编排
    - 路由跳转
    - 状态推进
    - 终止分支控制

不负责：
    - 具体规划
    - 具体调研
    - 具体蒸馏
    - 具体写作
  
### planner.py

负责：
    - 任务分解
    - 子任务优先级
    - 动态重规划
    - 收敛判断
    - 最终写作结构目标定义

不负责：
    - 具体 query 细化
    - 具体证据抽取
    - 最终正文生成
  
### researcher.py

负责：
    - 探索式检索
    - query reformulation
        - 多跳搜索
        - 来源发现
        - 原始材料采集

### distiller.py

负责：
    - 去噪
    - 结构化提取
    - 原子事实
    - 冲突检测
    - 证据质量评估
    - 摘要压缩
    - 如果没有单独 binder，则负责 section evidence pack 装配
  
### writer.py

负责：
基于 planner 给出的结构目标 + distiller/knowledge 给出的证据包进行写作
不自由重做 research
